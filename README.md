# Mexico Toy Store Sales Dashboard Analysis Report.

## Project Overview.

This Power BI dashboard provides a comprehensive analysis of the Mexico Toy Store business across different product categories, store locations, and time periods. The dashboard focuses on key business metrics including Sales, Cost, Profit, Units Sold, and Average Selling Price.


## Objective.

The primary objective of this dashboard is to answer the following business questions:
- Which product category generates the highest sales?
- Which category contributes the highest profit?
- Which store location performs best?
- Which store requires improvement?
- How have sales changed over time?
- Is profitability consistent across quarters?
- Which products have the highest average selling price?
- What is the overall profit margin?
- Which categories deserve additional investment?
- Which locations should receive expansion priority?

## Key Findings.

- Total Sales reached 14.44M with 4.01M profit.
- Toys are the top-performing category in both sales (5.1M) and profit (24.68% share).
- Electronics have the highest average price but the lowest profit contribution, suggesting opportunities to optimize pricing, costs, or demand.
- Downtown stores dominate performance, contributing 8.22M in sales (over half of total revenue).
- Sales show a gradual decline across quarters, while profit remains relatively stable due to effective cost management.
